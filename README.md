🧠 Overview

Nexus Chat is a modern, real-time chat application built for seamless communication. It combines powerful backend architecture with a clean, responsive UI to deliver a premium messaging experience.

📄 Defined as a real-time chat app with messaging, file sharing & status tracking

✨ Key Features

🚀 Real-Time Messaging – Instant communication using WebSockets
👥 Group & Private Chats – Flexible conversation types
📁 File Sharing – Send images, documents & media
🟢 Online Status Tracking – Know who's active
😊 Emoji Support – Express better
🔐 Authentication – Secure login via Firebase
⚡ Fast UI – Built with React + Vite
📊 Scalable Architecture – Ready for production
